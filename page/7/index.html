иҝҷдәӣйғҪеҸҜд»ҘиҜҙжҳҺпјҢе·Із»ҸжҲҗеҠҹеңЁе®№еҷЁеҶ…жҢӮиҪҪдәҶдёҖдёӘж–Үд»¶зі»з»ҹгҖӮиҖҢдё”пјҢеңЁе®ҝдё»жңәдёҠжҳҜж— жі•зңӢеҲ°иҝҷдёӘжҢӮиҪҪзӮ№зҡ„гҖӮеҲ°зӣ®еүҚдёәжӯўпјҢдёҖеҲҮйғҪжҳҜжҢүз…§йў„жңҹеҸ‘еұ•зҡ„гҖӮ

ж—ўз„¶еҸҜд»ҘеңЁе®№еҷЁеҶ…йғЁжҢӮиҪҪ /tmpпјҢйӮЈд№ҲзҺ°еңЁжқҘе°қиҜ•жҢӮиҪҪ /гҖӮ

йҰ–е…ҲпјҢеҮҶеӨҮдёҖдёӢеҝ…иҰҒзҡ„зҺҜеўғ:

  • bash: дҪңдёәе®№еҷЁзҡ„з¬¬дёҖдёӘиҝӣзЁӢпјҢе…Ғи®ёеңЁе®№еҷЁжү§иЎҢе…¶д»–жҢҮд»Ө
  • ls: и§ӮеҜҹе®№еҷЁеҶ…зҡ„ж–Үд»¶зі»з»ҹжҳҜеҗҰз¬ҰеҗҲйў„жңҹ
  • lib: еӯҳж”ҫ bash дёҺ ls еҝ…йЎ»зҡ„еҠЁжҖҒй“ҫжҺҘеә“

дёҺдёҠдёҖдёӘе®һйӘҢдёҚеҗҢзҡ„жҳҜпјҢзҺ°еңЁжңҹжңӣе®№еҷЁеҶ…зҡ„ж–Үд»¶зі»з»ҹдёҺе®ҝдё»жңәзӢ¬з«ӢгҖӮеҚіпјҢдҪҝз”ЁдёҚеҗҢзҡ„ж №зӣ®еҪ•гҖӮеӣ жӯӨпјҢеңЁд»Јз ҒеұӮйқўйңҖиҰҒе°Ҷд№ӢеүҚзҡ„ mount(2) зі»з»ҹи°ғз”ЁпјҢж”№еҸҳдёә chroot(2) зі»з»ҹи°ғз”ЁгҖӮ

з„¶еҗҺпјҢжү§иЎҢ ls еҸҜд»ҘзңӢеҲ°ж №зӣ®еҪ• / зҡ„ж–Үд»¶е°ұжҳҜд№ӢеүҚеңЁ root зӣ®еҪ•йў„е…ҲеҮҶеӨҮеҘҪзҡ„ж–Үд»¶:

з»јдёҠжүҖиҝ°пјҢеҰӮжһңеңЁ root зӣ®еҪ•дёӯдҝқеӯҳзҡ„жҳҜдёҖдёӘе®ҢжҲҗзҡ„ж“ҚдҪңзі»з»ҹпјҢйӮЈд№ҲпјҢе°ұеҸҜд»Ҙе®һзҺ°е®№еҷЁеҶ…зҡ„иҝӣзЁӢе°ұеҸҜд»ҘдҪҝз”ЁеҶ…йғЁзҡ„ /binпјҢ/lib зҡ„зі»з»ҹзҺҜеўғпјҢд»ҺиҖҢдёҺе®ҝдё»жңәпјҢдёҺе…¶д»–е®№еҷЁзӣёдә’йҡ”зҰ»зҡ„зӣ®зҡ„гҖӮ

UnionFS

еңЁ Cloud Foundry ж—¶д»ЈпјҢдёҠдә‘иҝҮзЁӢз№Ғзҗҗзҡ„ж №жң¬еҺҹеӣ еңЁдәҺ: жң¬ең°зҺҜеўғдёҺдә‘дё»жңәзҺҜеўғдёҚдёҖиҮҙгҖӮ

еңЁзӢӯд№үдёҠпјҢиҪҜд»¶зҡ„дҫқиө–жҢҮзј–зЁӢж—¶жүҖдҪҝз”Ёзҡ„з¬¬дёүж–№еә“пјӣе№ҝд№үдёҠпјҢдҫқиө–дёҚжӯўеҢ…жӢ¬зј–зЁӢдҪҝз”Ёзҡ„з¬¬дёүж–№еә“пјҢж“ҚдҪңзі»з»ҹеҗҢж ·д№ҹжҳҜиҪҜд»¶зҡ„дҫқиө–гҖӮ

йҖҡиҝҮдёҠйқўеҜ№ Mount Namespace жҠҖжңҜз®ҖеҚ•зҡ„е®һйӘҢпјҢе·Із»ҸйӘҢиҜҒдәҶеңЁе®№еҷЁеҶ…жү“еҢ…дёҖдёӘе®Ңж•ҙзҡ„ж“ҚдҪңзі»з»ҹдҪңдёәе®№еҷЁзҡ„ rootfs жҳҜеҸҜиЎҢзҡ„гҖӮеҲ°зӣ®еүҚдёәжӯўпјҢд»ҺжҠҖжңҜзҡ„и§’еәҰиҖҢиЁҖпјҢе·Із»Ҹи§ЈеҶідәҶе®№еҷЁйҡ”зҰ»зҡ„й—®йўҳгҖӮдҪҶд»ҺдҪҝз”Ёи§’еәҰпјҢжҲ–иҖ…иҜҙз”ЁжҲ·дҪ“йӘҢзҡ„и§’еәҰиҖҢиЁҖпјҢжҜҸж¬Ўжһ„е»әдёҖдёӘе®№еҷЁпјҢйғҪиҰҒжү“еҢ…дёҖд»Ҫж“ҚдҪңзі»з»ҹж–Үд»¶пјҢдјјд№Һе®һеңЁжҳҜдёҚдҫҝдәҺдҪҝз”ЁгҖӮ

жҜ”еҰӮпјҢејҖеҸ‘иҖ…дҪҝз”Ё Ubuntu ж“ҚдҪңзі»з»ҹзҡ„ ISO еҲ¶дҪңдәҶдёҖдёӘ rootfsпјҢе№¶дҫқж¬ЎдёәеҹәзЎҖпјҢе®үиЈ… Java зҺҜеўғпјҢиҝӣиҖҢйғЁзҪІ Java еә”з”ЁгҖӮеҰӮжһңпјҢеҸҰдёҖдёӘејҖеҸ‘иҖ…д№ҹжңүеҗҢж ·зҡ„йңҖжұӮпјҢжҲ–иҖ…еҗҢдёҖејҖеҸ‘иҖ…йңҖиҰҒйғЁзҪІеҸҰдёҖдёӘ Java еә”з”ЁпјҢжҳҫз„¶пјҢжңҖзҗҶжғіжғ…еҶөжҳҜиғҪеӨҹеӨҚз”Ёд№ӢеүҚе·Із»Ҹе®үиЈ…дәҶ Java зҡ„ rootfs зҺҜеўғпјҢиҖҢдёҚжҳҜйҮҚеӨҚдёҖйҒҚжөҒзЁӢгҖӮ

зӣҙи§Ӯзҡ„и§ЈеҶіеҠһжі•пјҢжһ„е»ә rootfs ж—¶пјҢжҜҸжү§иЎҢдёҖдёӘжңүж„Ҹд№үзҡ„ж“ҚдҪңд№ӢеҗҺпјҢйғҪз”ҹжҲҗдёҖдёӘж–°зҡ„ rootfsгҖӮд№ӢеҗҺпјҢе°ұеҸҜд»ҘйҖүжӢ©дёҖдёӘеҗҲйҖӮзҡ„ rootfs дҪңдёәеҹәзЎҖпјҢж·»еҠ ж–°зҡ„ж“ҚдҪңжһ„е»әзӣ®ж Ү rootfsгҖӮ

дҪҶиҝҷдёӘж–№жЎҲе№¶дёҚе®ҢзҫҺпјҢеҰӮжһңжҜҸж¬Ўжһ„е»әиҝҮзЁӢйғҪдә§з”ҹдёҖдёӘж–°зҡ„ rootfsпјҢжңҖеҗҺе°ҶдјҡеҜјиҮҙзі»з»ҹеҶ…жңүжһҒе…¶еӨҡзҡ„ rootfsгҖӮ

е№ёиҝҗзҡ„жҳҜпјҢиҝҷдёӘй—®йўҳе№¶дёҚйҡҫи§ЈеҶігҖӮе°Ҷй—®йўҳдёҖиҲ¬еҢ–пјҢеҹәдәҺж—ўжңү AпјҢжҜҸдёҖз§Қж“ҚдҪң F йғҪеҸҜд»Ҙдә§з”ҹе”ҜдёҖз»“жһң BгҖӮе№¶дё”пјҢB еҸҜд»ҘдҪңдёәдёӢдёҖиҪ®зҡ„иҫ“е…ҘгҖӮз»“жһңеҸ‘зҺ°пјҢиҝҷеғҸжһҒдәҶ git е’ҢеҢәеқ—й“ҫгҖӮ

Docker еңЁи®ҫи®Ў Docker й•ңеғҸж—¶д№ҹжҳҜдҪҝз”ЁдәҶзұ»дјјзҡ„ж–№жі•гҖӮд»–д»¬еј•е…ҘдәҶеұӮ (layer) жҰӮеҝөгҖӮз”ЁжҲ·еҲ¶дҪңй•ңеғҸзҡ„жҜҸдёҖжӯҘж“ҚдҪңпјҢйғҪдјҡз”ҹжҲҗдёҖдёӘеұӮпјҢе°Ҷ rootfs д»Һе…ЁйҮҸдҝқеӯҳпјҢж”№дёәдәҶеўһйҮҸдҝқеӯҳгҖӮ

д»Җд№ҲжҳҜ UnionFS

Docker й•ңеғҸзҡ„иҝҷз§Қе®һзҺ°ж–№ејҸпјҢдҫқиө–дәҺдёҖз§ҚеҸ«еҒҡ UnionFS зҡ„ж–Үд»¶зі»з»ҹгҖӮ

з®ҖеҚ•жқҘи®ІпјҢUnionFS е…Ғи®ёе°ҶеӨҡдёӘи®ҫеӨҮж–Үд»¶жҲ–зӣ®еҪ•жҢӮиҪҪеҲ°еҗҢдёҖдёӘзӣ®еҪ•дёҠпјҢе°ҶеӨҡдёӘи®ҫеӨҮзҡ„еҶ…е®№дҪңдёәж•ҙдҪ“еҜ№еӨ–еұ•зӨәпјҢжҲ–иҖ…е°ҶдёҖдёӘи®ҫеӨҮж–Үд»¶жҢӮиҪҪеҲ°дёҖдёӘе·Іжңүзҡ„зӣ®еҪ•дёҠгҖӮ

жҜ”еҰӮпјҢеңЁ Ubuntu 18.04 дёҠ:

дҪҝз”ЁиҒ”еҗҲжҢӮиҪҪзҡ„ж–№ејҸпјҢе°ҶиҝҷдёӨдёӘзӣ®еҪ•жҢӮиҪҪеҲ°дёҖдёӘе…¬е…ұзҡ„зӣ®еҪ• C дёҠ:

иҝҷж—¶еҸҜд»ҘзңӢеҲ°зӣ®еҪ• A дёҺзӣ®еҪ• B иў«еҗҲе№¶еҲ°дәҶзӣ®еҪ• C дёӯ

жӯӨж—¶пјҢеҰӮжһңдҝ®ж”№ C д№ҹдјҡеҸҚеә”еҲ°еҜ№еә”зҡ„ A жҲ– B дёӯгҖӮ

Docker ж”ҜжҢҒеӨҡз§ҚеӯҳеӮЁй©ұеҠЁпјҢдҪҶзӣ®еүҚй»ҳи®ӨдҪҝз”Ёзҡ„й©ұеҠЁдёә overlay2гҖӮ

Overlay2

жҺҘдёӢжқҘйҖҡиҝҮдёҖдёӘдҫӢеӯҗжқҘжҺўзҙў overlay2гҖӮ

йҰ–е…ҲпјҢеҗҜеҠЁдёҖдёӘе®№еҷЁ:

иҝҷдёӘе‘Ҫд»Өзҡ„еҗ«д№үжҳҜпјҢдҪҝз”Ё ubuntu:latest иҝҷдёӘ Docker й•ңеғҸжқҘиҝҗиЎҢдёҖдёӘе®№еҷЁгҖӮз„¶еҗҺпјҢдҪҝз”Ёе‘Ҫд»ӨжҹҘзңӢй•ңеғҸзҡ„з»ҶиҠӮ:

PaaS йЎ№зӣ®иў«еӨ§е®¶жҺҘзәізҡ„дёҖдёӘдё»иҰҒеҺҹеӣ пјҢе°ұжҳҜе®ғжҸҗдҫӣдәҶдёҖз§ҚеҗҚеҸ« еә”з”Ёжүҳз®Ў зҡ„иғҪеҠӣгҖӮеңЁеҪ“ж—¶пјҢиҷҡжӢҹжңәе’Ңдә‘и®Ўз®—е·Із»ҸжҳҜжҜ”иҫғжҷ®йҒҚзҡ„жҠҖжңҜе’ҢжңҚеҠЎпјҢдё»жөҒз”ЁжҲ·зҡ„жҷ®йҒҚз”Ёжі•пјҢе°ұжҳҜз§ҹдёҖжү№ AWS жҲ–иҖ… OpenStack зҡ„иҷҡжӢҹжңәпјҢз„¶еҗҺеғҸд»ҘеүҚз®ЎзҗҶзү©зҗҶжңҚеҠЎеҷЁйӮЈж ·пјҢз”Ёи„ҡжң¬жҲ–иҖ…жүӢе·Ҙзҡ„ж–№ејҸеңЁиҝҷдәӣжңәеҷЁдёҠйғЁзҪІеә”з”ЁгҖӮ

дҪҶжҳҜпјҢеңЁйғЁзҪІиҝҮзЁӢдёӯпјҢйҡҫе…Қдјҡзў°еҲ°дә‘з«ҜиҷҡжӢҹжңәе’Ңжң¬ең°зҺҜеўғдёҚдёҖиҮҙзҡ„й—®йўҳгҖӮжүҖд»ҘеҪ“ж—¶зҡ„дә‘и®Ўз®—жңҚеҠЎпјҢжҜ”зҡ„е°ұжҳҜи°ҒиғҪжӣҙеҘҪең°жЁЎжӢҹжң¬ең°жңҚеҠЎеҷЁзҺҜеўғпјҢиғҪеёҰжқҘжӣҙеҘҪзҡ„ дёҠдә‘ дҪ“йӘҢгҖӮиҖҢ PaaS ејҖжәҗйЎ№зӣ® Cloud Foundry е°ұжҳҜеҪ“ж—¶и§ЈеҶіиҝҷдёӘй—®йўҳзҡ„дёҖдёӘжңҖдҪіж–№жЎҲгҖӮ

дёҫдёӘж —еӯҗпјҢеҲӣе»әеҘҪиҷҡжӢҹжңәд№ӢеҗҺпјҢиҝҗз»ҙдәәе‘ҳеҸӘйңҖиҰҒеңЁиҝҷдәӣжңәеҷЁдёҠйғЁзҪІ Cloud Foundry AgentпјҢйҡҸеҗҺејҖеҸ‘иҖ…еҸӘиҰҒжү§иЎҢдёҖжқЎе‘Ҫд»Өе°ұиғҪжҠҠжң¬ең°зҡ„еә”з”ЁйғЁзҪІеҲ°дә‘дёҠ:

дәӢе®һдёҠпјҢCloud Foundry жңҖж ёеҝғзҡ„з»„д»¶е°ұжҳҜдёҖеҘ—еә”з”Ёзҡ„жү“еҢ…е’ҢеҲҶеҸ‘жңәеҲ¶гҖӮCloud Foundry дёәжҜҸз§Қдё»жөҒзј–зЁӢиҜӯиЁҖйғҪе®ҡд№үдәҶдёҖз§Қжү“еҢ…ж јејҸгҖӮ

cf push зҡ„дҪңз”ЁжҳҜжҠҠеә”з”Ёзҡ„еҸҜжү§иЎҢж–Үд»¶е’ҢеҗҜеҠЁи„ҡжң¬жү“иҝӣдёҖдёӘеҺӢзј©еҢ…еҶ…пјҢдёҠдј еҲ°дә‘дёҠ Cloud Foundry зҡ„еӯҳеӮЁдёӯгҖӮжҺҘзқҖпјҢCloud Foundry дјҡйҖҡиҝҮи°ғеәҰеҷЁйҖүжӢ©дёҖдёӘеҸҜд»ҘиҝҗиЎҢиҝҷдёӘеә”з”Ёзҡ„иҷҡжӢҹжңәпјҢз„¶еҗҺйҖҡзҹҘиҝҷдёӘжңәеҷЁдёҠзҡ„ Agent жҠҠеә”з”ЁеҺӢзј©еҢ…дёӢиҪҪдёӢжқҘеҗҜеҠЁгҖӮ

иҝҷж—¶еҖҷе…ій”®жқҘдәҶпјҢз”ұдәҺйңҖиҰҒеңЁдёҖдёӘиҷҡжӢҹжңәдёҠеҗҜеҠЁеҫҲеӨҡдёӘжқҘиҮӘдёҚеҗҢз”ЁжҲ·зҡ„еә”з”ЁпјҢCloud Foundry йҖҡиҝҮж“ҚдҪңзі»з»ҹзҡ„ cgroups е’Ң namespace жңәеҲ¶дёәжҜҸдёҖдёӘеә”з”ЁеҚ•зӢ¬еҲӣе»әдёҖдёӘз§°дҪң жІҷзӣ’ зҡ„йҡ”зҰ»зҺҜеўғпјҢз„¶еҗҺеңЁ жІҷзӣ’ дёӯеҗҜеҠЁиҝҷдәӣеә”з”ЁиҝӣзЁӢгҖӮиҝҷж ·пјҢе°ұе®һзҺ°дәҶжҠҠеӨҡдёӘз”ЁжҲ·зҡ„еә”з”Ёдә’дёҚе№Іж¶үең°еңЁиҷҡжӢҹжңәйҮҢжү№йҮҸең°гҖҒиҮӘеҠЁең°иҝҗиЎҢиө·жқҘзҡ„зӣ®зҡ„гҖӮиҖҢиҝҷдёӘ жІҷзӣ’ е°ұжҳҜжүҖи°“зҡ„ е®№еҷЁгҖӮ

иҖҢжң¬ж–Үзҡ„дё»и§’ Docker йЎ№зӣ®пјҢдёҺ Cloud Foundry зҡ„е®№еҷЁе№¶жІЎжңүжң¬иҙЁдёҠзҡ„е·®ејӮгҖӮеӣ жӯӨеңЁе®ғеҸ‘еёғеҗҺдёҚд№…пјҢCloud Foundry зҡ„йҰ–еёӯдә§е“Ғз»ҸзҗҶ James Bayer е°ұеңЁзӨҫеҢәйҮҢеҒҡдәҶдёҖж¬ЎиҜҰз»ҶеҜ№жҜ”пјҢе‘ҠиҜүз”ЁжҲ· Docker еҸӘжҳҜдёҖдёӘеҗҢж ·дҪҝз”Ё cgroups е’Ң namespace е®һзҺ°зҡ„ жІҷзӣ’ иҖҢе·ІпјҢжІЎжңүд»Җд№Ҳзү№еҲ«зҡ„й»‘з§‘жҠҖпјҢд№ҹдёҚйңҖиҰҒзү№еҲ«е…іжіЁгҖӮ

з„¶иҖҢпјҢзҹӯзҹӯеҮ дёӘжңҲпјҢDocker йЎ№зӣ®е°ұиҝ…йҖҹеҙӣиө·дәҶгҖӮе®ғзҡ„еҙӣиө·йҖҹеәҰеҰӮжӯӨд№Ӣеҝ«пјҢд»ҘиҮідәҺ Cloud Foundry д»ҘеҸҠжүҖжңүзҡ„ PaaS зӨҫеҢәиҝҳжІЎжқҘеҫ—еҸҠжҲҗдёәе®ғзҡ„з«һдәүеҜ№жүӢпјҢе°ұзӣҙжҺҘиў«е®Је‘ҠеҮәеұҖдәҶпјҢе Әз§° йҷҚз»ҙжү“еҮ»гҖӮ

Docker й•ңеғҸ

з©¶е…¶ж №жң¬еҺҹеӣ пјҢиҷҪз„¶ Docker дёҺ Cloud Foundry ж— и®әжҳҜеңЁж ёеҝғеҺҹзҗҶиҝҳжҳҜеңЁжҠҖжңҜе®һзҺ°дёҠеӨ§йғЁеҲҶзӣёеҗҢпјҢдҪҶжӯЈжҳҜиў«еӨ§е®¶еҝҪи§Ҷзҡ„йӮЈдёҖе°ҸйғЁеҲҶжҲҗдёә Docker зҡ„еҲ¶иғңжі•е®қвҖ”вҖ”Docker й•ңеғҸгҖӮ

Cloud Fondry жҲҗд№ҹвҖңжү“еҢ…вҖқпјҢиҙҘиҗ§вҖңжү“еҢ…вҖқгҖӮе…¶дёҖпјҢжҜҸз§ҚиҜӯиЁҖпјҢжҜҸз§ҚжЎҶжһ¶зҡ„жү“еҢ…ж–№ејҸйғҪдёҚз”ҡзӣёеҗҢпјҢз”ҡиҮідәҺжҜҸдёӘзүҲжң¬йғҪйңҖиҰҒжү“еҢ…пјӣе…¶дәҢпјҢиҷҪз„¶жү“еҢ…д№ӢеҗҺеҸҜд»ҘеңЁдә‘дёҠзӣҙжҺҘдҪҝз”ЁпјҢдҪҶеңЁд»Һжң¬ең°дёҠдә‘зҡ„иҝҮзЁӢдёӯпјҢеҸҜиғҪд»Қж—§йңҖиҰҒеҸҚеӨҚдҝ®ж”№гҖҒй…ҚзҪ®пјҢз”ҡиҮідәҺеңЁдёҚж–ӯиҜ•й”ҷдёӯдҪ“дјҡзҺ„еӯҰи°ғеҸӮгҖӮ

иҖҢ Docker й•ңеғҸеҚҙд»Һж №жң¬дёҠи§ЈеҶідәҶиҝҷдёҖй—®йўҳгҖӮDocker й•ңеғҸпјҢе®һйҷ…дёҠд№ҹжҳҜдёҖдёӘеҺӢзј©еҢ…пјҢдҪҶжҳҜеҶ…е®№еҚҙиҰҒжҜ” Cloud Foundry зҡ„дё°еҜҢзҡ„еӨҡгҖӮе®ғдёҚдҪҶеҢ…еҗ«еҸҜжү§иЎҢж–Үд»¶дёҺеҗҜеҠЁи„ҡжң¬пјҢжӣҙжҳҜеҢ…еҗ«дәҶдёҖдёӘе®Ңж•ҙзҡ„ж“ҚдҪңзі»з»ҹпјҢжүҖд»ҘиҝҷдёӘеҺӢзј©еҢ…зҡ„еҶ…е®№еҸҜд»ҘдёҺејҖеҸ‘зҺҜеўғгҖҒжөӢиҜ•зҺҜеўғзҡ„е®Ңе…ЁдёҖж ·гҖӮ

еҒҮи®ҫпјҢејҖеҸ‘ж—¶дҪҝз”Ё centos 8 дҪңдёәејҖеҸ‘зҺҜеўғпјҢжӯӨж—¶пјҢеҸӘйңҖиҰҒдҪҝз”Ё centos 8 зҡ„ iso иҝһеҗҢеҸҜжү§иЎҢж–Үд»¶дёҖиө·еҲ¶дҪңдёҖдёӘеҺӢзј©еҢ…пјҢйӮЈд№ҲпјҢж— и®әеңЁе“ӘйҮҢи§ЈеҺӢиҝҷдёӘеҺӢзј©еҢ…пјҢйғҪеҸҜд»ҘдёәеҸҜжү§иЎҢж–Үд»¶жҸҗдҫӣе®Ңе…ЁдёҖиҮҙзҡ„иҝҗиЎҢзҺҜеўғгҖӮ

TLS еҚҸи®®

дј иҫ“еұӮе®үе…ЁжҖ§еҚҸи®® (TLS: Transport Layer Security) еҸҠе…¶еүҚиә«е®үе…ЁеҘ—жҺҘеұӮ (SSL: Secure Sockets Layer) жҳҜдёҖз§Қе®үе…ЁеҚҸи®®пјҢзӣ®зҡ„жҳҜдёәдә’иҒ”зҪ‘йҖҡдҝЎжҸҗдҫӣе®үе…ЁеҸҠж•°жҚ®е®Ңж•ҙжҖ§дҝқйҡңгҖӮ

SSL еҢ…еҗ«и®°еҪ•еұӮ (Record Layer) е’Ңдј иҫ“еұӮпјҢи®°еҪ•еұӮеҚҸи®®зЎ®е®ҡдј иҫ“еұӮж•°жҚ®зҡ„е°ҒиЈ…ж јејҸгҖӮдј иҫ“еұӮе®үе…ЁеҚҸи®®дҪҝз”Ё X.509 и®ӨиҜҒпјҢз„¶еҗҺеҲ©з”ЁйқһеҜ№з§°еҠ еҜҶжј”з®—жқҘеҜ№йҖҡдҝЎж–№еҒҡиә«д»Ҫи®ӨиҜҒпјҢд№ӢеҗҺдәӨжҚўеҜ№з§°еҜҶй’ҘдҪңдёәдјҡи°ҲеҜҶй’Ҙ (Session key)гҖӮиҝҷдёӘдјҡи°ҲеҜҶй’ҘжҳҜз”ЁжқҘе°ҶйҖҡдҝЎдёӨж–№дәӨжҚўзҡ„ж•°жҚ®еҒҡеҠ еҜҶпјҢдҝқиҜҒдёӨдёӘеә”з”Ёй—ҙйҖҡдҝЎзҡ„дҝқеҜҶжҖ§е’ҢеҸҜйқ жҖ§пјҢдҪҝе®ўжҲ·дёҺжңҚеҠЎеҷЁеә”з”Ёд№Ӣй—ҙзҡ„йҖҡдҝЎдёҚиў«ж”»еҮ»иҖ…зӘғеҗ¬гҖӮ

HTTPS еҚҸи®®

и¶…ж–Үжң¬дј иҫ“е®үе…ЁеҚҸи®® (HTTPS: HyperText Transfer Protocol SecureпјҢеёёз§°дёәHTTP over TLSгҖҒHTTP over SSLжҲ–HTTP Secure) жҳҜдёҖз§ҚйҖҡиҝҮи®Ўз®—жңәзҪ‘з»ңиҝӣиЎҢе®үе…ЁйҖҡдҝЎзҡ„дј иҫ“еҚҸи®®гҖӮHTTPS з»Ҹз”ұ HTTP иҝӣиЎҢйҖҡдҝЎпјҢдҪҶеҲ©з”Ё SSL/TLS жқҘеҠ еҜҶж•°жҚ®еҢ…гҖӮHTTPS ејҖеҸ‘зҡ„дё»иҰҒзӣ®зҡ„пјҢжҳҜжҸҗдҫӣеҜ№зҪ‘з«ҷжңҚеҠЎеҷЁзҡ„иә«д»Ҫи®ӨиҜҒпјҢдҝқжҠӨдәӨжҚўж•°жҚ®зҡ„йҡҗз§ҒдёҺе®Ңж•ҙжҖ§гҖӮ

и®ӨиҜҒиҝҮзЁӢ

еҚ•еҗ‘и®ӨиҜҒ

еңЁи®ҝй—®еӨ§еӨҡж•°зҪ‘з«ҷ (google, facebook) ж—¶пјҢдјҡдҪҝз”ЁеҚ•еҗ‘и®ӨиҜҒзҡ„ж–№ејҸгҖӮе®ўжҲ·з«Ҝ(жөҸи§ҲеҷЁ)дјҡйӘҢиҜҒжңҚеҠЎз«ҜиҜҒд№Ұзҡ„еҗҲжі•жҖ§пјҢиҝҮзЁӢеҰӮдёӢпјҡ

  1. е®ўжҲ·з«ҜеҸ‘иө·е»әз«Ӣ HTTPS иҝһжҺҘиҜ·жұӮпјҢе°Ҷ SSL еҚҸи®®зүҲжң¬зҡ„дҝЎжҒҜеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁз«Ҝ
  2. жңҚеҠЎеҷЁз«Ҝе°Ҷжң¬жңәзҡ„е…¬й’ҘиҜҒд№Ұ (server.crt) еҸ‘йҖҒз»ҷе®ўжҲ·з«Ҝ
  3. е®ўжҲ·з«ҜиҜ»еҸ–е…¬й’ҘиҜҒд№Ұ (server.crt) еҸ–еҮәжңҚеҠЎз«Ҝе…¬й’Ҙ

йҡҸзқҖж•°жҚ®еә“еҸҳеҫ—и¶ҠжқҘи¶Ҡз№ҒеҝҷпјҢд»»дҪ•ж•°жҚ®еә“еә”з”ЁзЁӢеәҸжңҖз»ҲйғҪдјҡиҫҫеҲ°зЎ¬д»¶жһҒйҷҗгҖӮ DBAеҝ…йЎ»иҜ„дј°жҳҜеҗҰжңүеҸҜиғҪи°ғж•ҙеә”з”ЁзЁӢеәҸжҲ–йҮҚж–°й…ҚзҪ®жңҚеҠЎеҷЁд»ҘйҒҝе…Қиҝҷдәӣз“¶йўҲпјҢжҲ–иҖ…жҳҜеҗҰйңҖиҰҒжӣҙеӨҡзҡ„зЎ¬д»¶иө„жәҗгҖӮ зі»з»ҹз“¶йўҲйҖҡеёёжқҘиҮӘд»ҘдёӢжқҘжәҗпјҡ

  • зЈҒзӣҳжҗңеҜ»гҖӮ зЈҒзӣҳжҹҘжүҫж•°жҚ®йңҖиҰҒиҠұиҙ№ж—¶й—ҙгҖӮ еҜ№дәҺзҺ°д»ЈзЈҒзӣҳпјҢжӯӨж“ҚдҪңзҡ„е№іеқҮж—¶й—ҙйҖҡеёёе°ҸдәҺ10жҜ«з§’пјҢеӣ жӯӨзҗҶи®әдёҠжҲ‘д»¬еҸҜд»Ҙжү§иЎҢ100ж¬ЎжҗңзҙўгҖӮ иҝҷж®өж—¶й—ҙйҡҸзқҖж–°зЈҒзӣҳзҡ„дҪҝз”ЁиҖҢзј“ж…ўж”№е–„пјҢе№¶дё”еҫҲйҡҫдёәеҚ•дёӘиЎЁиҝӣиЎҢдјҳеҢ–гҖӮ дјҳеҢ–еҜ»йҒ“ж—¶й—ҙзҡ„ж–№жі•жҳҜе°Ҷж•°жҚ®еҲҶеҸ‘еҲ°еӨҡдёӘзЈҒзӣҳдёҠгҖӮ
  • зЈҒзӣҳиҜ»еҶҷгҖӮ еҪ“зЈҒзӣҳдҪҚдәҺжӯЈзЎ®зҡ„дҪҚзҪ®ж—¶пјҢжҲ‘д»¬йңҖиҰҒиҜ»еҸ–жҲ–еҶҷе…Ҙж•°жҚ®гҖӮ еҜ№дәҺзҺ°д»ЈзЈҒзӣҳпјҢдёҖдёӘзЈҒзӣҳиҮіе°‘еҸҜжҸҗдҫӣ10вҖ“20MB / sзҡ„еҗһеҗҗйҮҸгҖӮ дёҺжҹҘжүҫзӣёжҜ”пјҢдјҳеҢ–иө·жқҘжӣҙе®№жҳ“пјҢеӣ дёәжӮЁеҸҜд»Ҙд»ҺеӨҡдёӘзЈҒзӣҳе№¶иЎҢиҜ»еҸ–гҖӮ
  • CPUе‘ЁжңҹгҖӮ еҪ“ж•°жҚ®дҪҚдәҺдё»еӯҳеӮЁеҷЁдёӯж—¶пјҢжҲ‘д»¬еҝ…йЎ»еҜ№е…¶иҝӣиЎҢеӨ„зҗҶд»ҘиҺ·еҫ—з»“жһңгҖӮ дёҺеҶ…еӯҳйҮҸзӣёжҜ”пјҢжӢҘжңүиҫғеӨ§зҡ„иЎЁжҳҜжңҖеёёи§Ғзҡ„йҷҗеҲ¶еӣ зҙ гҖӮ дҪҶжҳҜеҜ№дәҺе°ҸжЎҢеӯҗпјҢйҖҹеәҰйҖҡеёёдёҚжҳҜй—®йўҳгҖӮ
  • еҶ…еӯҳеёҰе®ҪгҖӮ еҪ“CPUйңҖиҰҒзҡ„ж•°жҚ®и¶…еҮәCPUзј“еӯҳзҡ„е®№йҮҸж—¶пјҢдё»еҶ…еӯҳеёҰе®Ҫе°ҶжҲҗдёәз“¶йўҲгҖӮ еҜ№дәҺеӨ§еӨҡж•°зі»з»ҹжқҘиҜҙпјҢиҝҷжҳҜдёҖдёӘдёҚеёёи§Ғзҡ„з“¶йўҲпјҢдҪҶжҳҜиҰҒж„ҸиҜҶеҲ°иҝҷдёҖзӮ№гҖӮ
